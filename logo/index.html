Fonts & browser consistency

On this page the logo renders with Google Fonts loaded in the browser (Martel Sans, Averia Serif Libre). The XP monogram uses Palatino / Georgia — both are pre-installed on every desktop OS and render consistently.

When you use the downloaded SVG as an <img>, browsers block external font requests inside SVG files. The XP will still render correctly (Palatino/Georgia is system-installed). RESONANT will fall back to system-ui. To guarantee RESONANT's exact weight & spacing, embed the SVG inline in your HTML instead of via <img> — the page's loaded fonts then apply to it.
